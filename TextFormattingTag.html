Italics Tag

1.It is Container tag.
2.It is inline level elment.
3.It is used to Display content in Italic style.

Bold Tag

1.It is Container tag.
2.It is inline level elment.
3.It is used to Display content in Bold style.

Underline Tag

1.It is Container tag.
2.It is inline level elment.
3.It is used to underline the text.

Superscript Tag

1.It is Container tag.
2.It is inline level elment.
3.It is used to display text as Superscript.
e.g : a2+ b 2

Subscript Tag

1.It is Container tag.
2.It is inline level elment.
3.It is used to display text as Subscript.
e.g : log e

Code Tag

1.It is Container tag.
2.It is inline level elment.
3.It is used indicate that the section of text is code.
e.g : 
                  public static void main(String [] args)
                  {
                     System.out.println("Hello World !")
                  }
